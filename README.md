# Vue3 学习 demo
一个 Vue3 的 Demo，语法基于 Typescript，全面使用了 Vue3 相关的生态，如 Vite、Vuex、Vue-router、Element-plus 等

## 计划
- [x] Vite  目前存在较多 bug，实际生产中不便使用
- [x] Typescript
- [x] Vue-router
- [x] Vuex
- [x] Axios
- [ ] Swiper  适配不佳，会出现栈溢出的问题，暂时无法使用
- [x] Echarts
- [x] Element-plus

## 开发
```bash
yarn
yarn dev
```
或者使用 npm
```bash
npm i
npm run dev
```
